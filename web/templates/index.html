ASL Leap Motion Help

Introduction

ASL LeapMotion teaches newcomers the alphabet of American Sign Language, also called ASL. The user is shown a letter of the finger alphabet and then he has to try to reproduce the letter shown by hand. However, this is not as easy as it sounds at first glance, because there is a artificial intelligence. This AI has to be convinced that the hand-formed letter is also the one displayed on the surface.

How To

First of all you have to check if the Leap Motion controller is installed and set up correctly. When you visit the website of the application a welcome screen is initially displayed. With the "Start" button it disappears and the actual learning interface becomes visible.

(1) The picture on the left side shows the finger alphabet letter which is currently being learned. The letter has to be reproduced by hand.

(2) The picture on the right side or the question mark if no hand is recognized, shows the letter which the AI believes to recognize at the moment.

(3) A vertical progress bar is shown between the two images. This indicates how well the current letter is recognized by the AI. If this is the case, a success window appears and shows how well the letter was recognized by the AI. By pressing the next button the window closes and the next letter for learning is displayed automatically.

(4) A status text is displayed below, which adapts dynamically.

(5) There is a help button in the lower right corner. You have probably found it already.

FAQ

"No hand detected" is still displayed although one hand is above the sensor ?

First check if the Leap Motion Sensor is properly connected (green LED is on) and the Leap Motion Control Panel is started. If the Leap Motion Software tray icon is also green, the sensor should work correctly. Sometimes it helps to reconnect the sensor to the computer.
